# 星露谷物语像素田园前台重设计规范

## 目标

在保留现有 Express API、后台管理、图片上传、搜索、分页、攻略文章和三个实用工具的前提下，将公开前台统一重构为已确认定稿图中的像素田园风格。

## 视觉基准

- 唯一总览基准：`stardew-ui-final-approved-draft.png`
- 首页局部约束：
  - “浏览完整图鉴”和“阅读攻略”作为一个按钮组，中心线与主标题中心线一致。
  - 按钮组与说明文字、四张统计卡均保留清晰垂直间距。
  - 热门攻略固定展示 6 张完整卡片；桌面端等宽、等高、底边对齐。
  - 热门攻略卡片不显示版本号。
- 颜色与材质：
  - 深木棕导航和框架。
  - 奶油米黄内容面板。
  - 金黄色标题、按钮和选中状态。
  - 草绿、湖蓝用于状态和轻量点缀。
  - 正文使用深棕色。
- 所有可见图标使用项目现有本地像素图片，不使用表情符号或临时占位图。

## 页面结构

### 首页

木质导航、农场首屏、中文主标题、居中操作按钮、四张统计卡、六张快捷分类卡、三个工具入口、六张热门攻略卡和页脚。

### 攻略分类页

`#library` 无 `dataset` 参数时显示分类总览。九个主分类按三列网格排列，“完整图鉴索引”作为独立入口保留。

### 图鉴列表页

`#library?dataset=<slug>` 显示左侧木质分类栏、顶部搜索和排序工具栏、右侧像素物品网格、分页与表格/卡片视图切换。

### 物品详情弹窗

`#entry/<datasetId>/<slug>` 在对应图鉴列表上方打开居中木框弹窗。左侧显示物品大图，右侧提供“获取方式 / 用途 / 售价”标签页；关闭后返回原分类列表。

### 攻略详情页

顶部木质标题牌，下方奶油色正文面板，右侧悬浮目录。移动端目录折叠到正文上方。

## 响应式

- 大于 1100px：完整桌面布局。
- 760px 到 1099px：分类和攻略卡片改为两至三列，图鉴侧栏变为顶部折叠区。
- 小于 760px：单列内容、横向可滚动导航、按钮组允许换行、弹窗改为上下布局。

## 广告预留

首页热门攻略后、图鉴列表中段和攻略正文中段保留隐藏广告容器。未配置广告时使用 `hidden`，不产生空白或影响布局。

## 数据与兼容性

- API 数据结构保持不变。
- 后台管理页面不纳入本次视觉重构。
- 图片继续从 `/assets/game/` 和 `/uploads/` 加载。
- 用户已保存的社区中心进度不变。

## 验收标准

- 五类页面与定稿图保持一致的结构、层级和田园像素风格。
- 首页两个操作按钮严格居中。
- 六张热门攻略卡片完整、等高、底边一致，无版本号。
- 所有交互控件可用，键盘焦点可见。
- 现有测试全部通过，并新增前台视觉契约测试。
- 完成桌面和移动端浏览器截图检查。
